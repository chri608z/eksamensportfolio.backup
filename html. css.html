Velkommen til web processen

HTML

HTML står for Hyper Text Markup Language, som er med til at strukturer indholdet på hjemmesiden bliver bygget op. Men det er ikke det eneste der hjælper HTML, der bliver også brugt nogle forskellige elementer, disse HTML-elementer består af tags og i disse tags kan man skrive tekst eller andet data som er relevant for ens hjemmeside.

CSS

CSS er et computersprog som bruges til at beskrive hvordan man ønsker et HTLM-side bliver præsenteret. CSS står for Cascading Style Sheets. På en måde så kan man sige at CSS farvelægger og visualiserer ens HTML-sider. Layoutet for ens sider når man bruger CSS er at man kan bestemmer skrifttyper, tekststørrelser, farver, marginer, baggrundsbilleder, farven på ens baggrund osv.
